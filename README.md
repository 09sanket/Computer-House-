# Computer House - Modern React.js Project

A modern, responsive website built with React.js, Three.js, Framer Motion, and Tailwind CSS.

## Tech Stack

- **React.js** (latest) - UI library
- **Vite** - Build tool and dev server
- **Three.js** with `@react-three/fiber` and `@react-three/drei` - 3D graphics
- **Framer Motion** - Smooth UI animations
- **Tailwind CSS** - Utility-first CSS framework
- **React Router DOM** - Multi-page routing

## Project Structure

```
src/
 ├─ assets/           // Images, videos, icons
 ├─ components/       // Reusable components
 ├─ pages/            // Home, About, Services, Products, Repairs, Contact
 ├─ three/            // Three.js 3D components
 ├─ layout/           // Navbar, Footer
 ├─ App.jsx           // Main app component with routing
 └─ main.jsx          // Entry point
```

## Theme

- **Background**: Clean white
- **Accent Gradients**: Blue → Cyan → Purple
- **Text Color**: Cool blue shades
- **Effects**: Modern glassmorphism + soft shadows

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
